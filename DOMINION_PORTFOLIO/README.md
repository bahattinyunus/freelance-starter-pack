# Portfolyo Şablonu

İşlerinizi sergilemek için basit, temiz ve duyarlı (responsive) bir portfolyo şablonu.

## Nasıl Kullanılır?
1.  `index.html` dosyasını bir metin editöründe açın.
2.  `[Adınız Soyadınız]`, `you@example.com` ve proje detaylarını kendi bilgilerinizle değiştirin.
3.  Daha fazla özelleştirme için `<style>` etiketine kendi CSS kodlarınızı ekleyebilir veya harici bir stil dosyası bağlayabilirsiniz.
4.  GitHub Pages, Netlify veya Vercel üzerinde ücretsiz yayınlayın.

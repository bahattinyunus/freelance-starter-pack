# Portfolio Template

This is a simple, clean, and responsive portfolio template to showcase your work.

## How to Use
1.  Open `index.html` in a text editor.
2.  Replace `[Your Name]`, `you@example.com`, and project details with your own information.
3.  Add your own CSS in the `<style>` tag or link an external stylesheet for more customization.
4.  Deploy to GitHub Pages, Netlify, or Vercel.
